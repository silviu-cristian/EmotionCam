EmotionCam User Manual

Local visible-expression estimates with privacy-first personalized calibration

Version 1.0.0 · June 16, 2026

Important: EmotionCam estimates visible expressions. It does not know true emotions and is not for medical, psychological, legal, hiring, security, or lie-detection use.

Introduction

EmotionCam is a Windows desktop app for one person using a laptop webcam. It processes frames locally, shows smoothed visible-expression estimates, supports personalized calibration, can run in the Windows tray, saves optional metadata-only logs, and can show local Statistics.

Privacy and Safety

  • No identity recognition, face matching, accounts, telemetry, or automatic recording.
  • Normal use does not save images or video.
  • Calibration saves local normalized feature data and labels.
  • Daily email summaries are off by default and send text only if explicitly enabled.

Installation

  1. Download EmotionCam_Setup.exe from the latest GitHub Release.
  2. Run the installer and launch from Start Menu or desktop shortcut.
  3. Allow Windows camera access for desktop apps.

From source: create a Python 3.11/3.12 virtual environment, install requirements.txt, then run python -m app.main.

First Launch

The startup screen shows privacy and approximation notices, the metadata-history choice, Start Camera, Settings, and Exit.

Main Dashboard

The dashboard contains the camera preview, colored face rectangle, current likely expression, group, confidence, message, timeline, detection mode, profile status, Statistics, Settings, Open Logs, Run in background, and Exit controls.

Blue means positive, orange means negative, and gray means neutral/unknown/no face.

Camera Controls

  1. Click Start Camera.
  2. Use Pause Analysis to keep preview while classification pauses.
  3. Click Stop Camera to release the webcam and show the black Camera stopped screen.

Expression Detection

Positive labels include happy, smile, smile_small, smile_big, laughing, amused, and surprised. Negative labels include sad, angry, fearful, disgusted, tired, bored, frustrated, and concerned. Neutral-group labels include neutral, focused, confused, skeptical, thinking, relaxed, unknown, low_confidence, and no_face.

Personalized Calibration

Open calibration from the dashboard or Settings. Choose a target directly from the dropdown, use the local example graphic and match helper, then use Previous, Start Capture, Re-capture, Next, and Finish.

The old Skip button and separate Label Unknown Expression feature were removed.

Settings, Themes, and Spinner Arrows

Choose Appearance > Theme > Dark/Light and Save. Numeric inputs can be typed or changed with visible up/down arrows.

Local Statistics

Click Statistics to view selected-day balance, daily group timeline, expression counts, last-7-days balance, most frequent expression, analyzed time estimate, and positive/serious periods. Charts use local metadata logs only.

User Profile and Optional Daily Email

User name and email are optional. Daily email summaries are disabled by default and send local statistics text only. SMTP passwords are not stored in profile JSON and use secure credential storage when available.

Background Mode, Logs, and Local Data

Run in background keeps local analysis active in the tray. Logs are metadata-only and stored under %LOCALAPPDATA%\EmotionCam\logs.

Demo Walkthrough

  1. Launch and show the icon/startup screen.
  2. Start camera; show positive blue rectangle.
  3. Stop camera; show Camera stopped.
  4. Open Settings; show arrows and themes.
  5. Open calibration; show dropdown, graphic, match helper, capture, recapture, navigation, and finish.
  6. Open Statistics, User Profile, Email Summary, Background, Logs, and Exit.

Troubleshooting and Limitations

IssueTry
Camera unavailableClose other webcam apps and allow desktop camera access.
Poor detection/calibrationImprove lighting, face the camera, hold still, and calibrate.
No Statistics dataEnable logging, run analysis, refresh, and select a date with entries.

Estimates remain approximate; calibration improves but cannot guarantee accuracy.

Uninstall and Data Removal

Uninstall from Windows Installed apps. Fully exit EmotionCam and delete %LOCALAPPDATA%\EmotionCam to remove settings, logs, profiles, and optional debug frames.
