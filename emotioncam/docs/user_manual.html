EmotionCam User Manual

Local visible-expression estimates with privacy-first personalized calibration

Version 1.0.0 · June 15, 2026

Important: EmotionCam estimates visible expressions. It does not know or guarantee true emotions and is not for medical, psychological, legal, hiring, security, or lie-detection use.

Introduction

EmotionCam is a Windows desktop app for one person using a laptop webcam. It processes frames locally, shows smoothed visible-expression estimates, supports personalized calibration, can run in the Windows tray, and can optionally save metadata-only logs.

Privacy and Safety

  • No cloud calls, identity recognition, face matching, accounts, telemetry, or automatic recording.
  • Normal use does not save images or video.
  • Calibration saves local normalized landmark features and labels.
  • Raw calibration frames are saved only if the explicit debugging option is enabled.

Installation

  1. Run EmotionCam-Setup-1.0.0.exe.
  2. Complete installation and launch from Start Menu or desktop shortcut.
  3. Allow Windows camera access for desktop apps.

From source: create a Python 3.11/3.12 virtual environment, install requirements.txt, then run python -m app.main.

Camera Controls

  1. Click Start Camera.
  2. Use Pause Analysis to keep the preview while classification pauses.
  3. Click Stop Camera to release the webcam and show the black Camera stopped screen.
  4. Click Start Camera to restart.

Expression Detection

Positive labels include happy, smile, smile_small, smile_big, laughing, amused, and surprised. Negative labels include sad, angry, fearful, disgusted, tired, bored, frustrated, and concerned. Neutral-group labels include neutral, focused, confused, skeptical, thinking, relaxed, unknown, low_confidence, and no_face.

Confidence and smoothing reduce flicker but cannot guarantee correctness. Lighting, blur, distance, occlusion, and head angle affect results.

Personalized Calibration

Open calibration from the dashboard or Settings. Choose a target directly from the dropdown, use the local example graphic and live match helper, then use Previous, Start Capture, Re-capture, Next, and Finish.

The match check is only a helper and never blocks capture. Re-capture replaces the target's prior batch. The old Skip button and separate Label Unknown Expression feature were removed.

Settings, Themes, and Spinner Arrows

Choose Appearance > Theme > Dark/Light and Save. The selection is stored locally and applies app-wide.

All numeric inputs are QSpinBox/QDoubleSpinBox controls. Values can be typed or changed with clearly visible, clickable up/down arrows in both themes.

Background Mode

Run in background keeps camera analysis and optional logging active locally. The tray menu can Show EmotionCam, Stop Camera, or Exit. Optional stable-group popups are cooldown-controlled and do not diagnose the user.

Logs and Local Data

DataPath
Config%LOCALAPPDATA%\EmotionCam\config.json
Logs%LOCALAPPDATA%\EmotionCam\logs\expression_history.jsonl
Profile%LOCALAPPDATA%\EmotionCam\profile\expression_profile.json

Logs contain metadata only and explicitly exclude frame/image fields.

Local Statistics

Click Statistics on the dashboard to view selected-day expression balance, daily group timeline, label counts, last-7-days balance, most frequent expression, approximate analyzed time, and most positive/serious periods. Charts use local metadata logs only and support dark/light themes.

User Profile and Optional Daily Email

User name and email are optional and stored locally in %LOCALAPPDATA%\EmotionCam\profile\user_profile.json. Daily email summaries are off by default and send statistics text only, never frames or images.

The default-mail-client option creates a manual draft. SMTP automatic sending runs only after explicit opt-in, only when logs exist, and at most once per date. Passwords are never stored in profile JSON and use secure credential storage when available.

Troubleshooting and Limitations

IssueTry
Camera unavailableClose other webcam apps and allow desktop camera access.
Poor detection/calibrationImprove lighting, face the camera, hold still, and calibrate.
Arrows/theme unchangedSave Settings and ensure the newly built app is running.
No trayEmotionCam minimizes to the taskbar.

Estimates remain approximate; calibration improves but cannot guarantee accuracy.

Uninstall and Data Removal

Uninstall from Windows Installed apps. Fully exit EmotionCam and delete %LOCALAPPDATA%\EmotionCam to remove settings, logs, profile features, and optional debug frames.
