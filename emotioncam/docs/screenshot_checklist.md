# EmotionCam Screenshot Checklist

The startup and Settings examples are camera-free local Qt renders. Camera, calibration,
tray, and logs examples are clearly labeled generated placeholders. Replace them with real
screens on an interactive Windows desktop using the exact names below. Do not expose
personal notifications or unrelated files.

## Capture Rules

1. Keep raw calibration debug-image storage off.
2. Capture both dark and light theme examples.
3. Verify spinner arrows are visible before capturing Settings.
4. Do not manufacture a negative expression merely to trigger a popup.
5. Replace placeholder notes in the manual after adding real screenshots.
