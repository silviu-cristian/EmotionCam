Start the EmotionCam Demo Here

Offline presenter guide for EmotionCam 1.0.0

Visible-expression estimates, local processing, personalized calibration, and no cloud calls.

Before the room fills: close other webcam apps, allow Windows camera access, use steady lighting, and open EmotionCam. In the app, this page opens from Help > Start Demo Guide.

Quick Start

1. Open and explain privacy

Show the simplified lens icon, startup privacy notice, and optional metadata-history checkbox.

2. Show live detection

Click Start Camera, hold neutral, then smile. Point out the smoothed label and blue positive-group rectangle.

3. Show controls and themes

Open Settings, click the visible spinner arrows, type a value, then switch Dark to Light and save.

4. Show calibration

Use the expression dropdown, local example graphic, match helper, Start Capture, Re-capture, Previous, Next, and Finish.

Complete Demo Checklist

1
Launch EmotionCam and show the app icon and startup screen.

Demo-Day Troubleshooting

  • Camera unavailable: close Camera/Teams/Zoom/browser calls and verify permissions.
  • No face or poor calibration quality: improve lighting, face the camera, and hold still.
  • No tray: EmotionCam minimizes to the taskbar as a fallback.
  • Old arrows/theme: close any old packaged build and start the newly built application.
  • Do not force a negative expression for popups; explain their stable-group cooldown behavior.
EmotionCam processes visible-expression estimates locally. It does not know true emotions.
