# EmotionCam v1.1.3 AI-enabled Test Connection Hotfix Release Notes

EmotionCam v1.1.3-ai improves the External AI settings experience.

## What Changed

- The **Test Connection** button now visibly changes to **Testing...** while
  the OpenAI connection test is running.
- The Settings dialog now shows a clear status line for the test result.
- The app now shows a success or failure popup after the test completes.
- Missing API key errors now reset the button and show the reason in the
  Settings dialog.
- Added tests for AI connection test progress and result feedback.

## Privacy

This hotfix does not change privacy behavior. External AI remains off by default
and still requires explicit consent and an API key before any image can be sent.

## Installer

Recommended release asset:

```text
EmotionCam_Setup_v1.1.3_AI.exe
```
