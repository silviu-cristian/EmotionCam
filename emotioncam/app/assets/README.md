# EmotionCam Assets

`icon.png` is the official source application artwork. `icon.ico` contains
16, 32, 48, 64, 128, and 256 pixel Windows icon sizes and is used
by the window, taskbar, tray, PyInstaller, Inno Setup, and shortcuts.

`expression_examples.json` contains local vector-graphic specifications rendered
inside the calibration wizard. No example visuals are fetched at runtime.

`face_landmarker.task` is the official MediaPipe Face Landmarker model bundled
for fully local landmark and blendshape processing. It performs no network calls.
